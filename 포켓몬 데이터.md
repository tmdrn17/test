## <a id="타입"></a>1. 타입
포켓몬스터의 모든 포켓몬과 기술은 고유의 '타입'을 가집니다. 4세대 하트골드/소울실버에는 총 17개의 타입이 존재하며, 배틀 시 대미지 계산의 핵심 기준이 됩니다.

### <a id="타입-노말"></a>1.1. 노말
가장 기본적인 타입으로, 특징적인 상성 관계가 적어 무난하게 사용됩니다.
### <a id="타입-불꽃"></a>1.2. 불꽃
뜨거운 불꽃을 다루는 타입입니다. 공격적인 상성이 우수하며 풀, 얼음, 벌레, 강철 타입에게 강력합니다.
### <a id="타입-물"></a>1.3. 물
바다와 강을 상징하는 타입입니다. 약점이 적고 방어 및 공격 밸런스가 매우 뛰어난 타입입니다.
### <a id="타입-풀"></a>1.4. 풀
자연을 상징하는 타입입니다. 약점이 5개로 많지만, 다양한 상태이상기와 흡수기를 보유하고 있습니다.
### <a id="타입-전기"></a>1.5. 전기
짜릿한 전류를 사용하는 타입입니다. 약점이 땅 타입 하나뿐이며 스피드가 빠른 포켓몬이 많습니다.
### <a id="타입-얼음"></a>1.6. 얼음
차가운 얼음과 눈을 다룹니다. 방어 상성은 취약하지만 드래곤, 비행 등을 저격하는 최고의 공격 타입입니다.
### <a id="타입-격투"></a>1.7. 격투
육체적 강함을 다루는 타입입니다. 노말, 얼음, 바위, 악, 강철 등의 약점을 찌를 수 있는 핵심 공격 타입입니다.
### <a id="타입-독"></a>1.8. 독
독소와 오염을 다루는 타입입니다. 독 상태이상을 걸거나 방어적으로 우수한 면모를 보입니다.
### <a id="타입-땅"></a>1.9. 땅
대지의 힘을 이용하는 타입입니다. 전기 타입을 완벽히 무효화하며, 위력적인 기술인 '지진'이 유명합니다.
### <a id="타입-비행"></a>1.10. 비행
하늘을 나는 포켓몬들의 타입입니다. 땅 타입 기술을 무효화하며 주로 다른 타입과 복합으로 존재합니다.
### <a id="타입-에스퍼"></a>1.11. 에스퍼
초능력을 사용하는 타입입니다. 높은 특수공격력을 바탕으로 상대방을 압도합니다.
### <a id="타입-벌레"></a>1.12. 벌레
곤충을 모티브로 한 타입입니다. 초반에 빠르게 성장하는 포켓몬이 많으며, 유턴 등의 유용한 기술이 있습니다.
### <a id="타입-바위"></a>1.13. 바위
단단한 암석의 타입입니다. 모래바람 날씨에서 특수방어가 1.5배 상승하는 특징이 있습니다.
### <a id="타입-고스트"></a>1.14. 고스트
유령과 영혼의 타입입니다. 노말과 격투라는 대중적인 두 가지 타입을 무효화하는 유니크한 상성을 가집니다.
### <a id="타입-드래곤"></a>1.15. 드래곤
전설 속 용의 타입입니다. 반감하는 타입이 많고 종족값이 높은 강력한 포켓몬들이 대거 포진해 있습니다.
### <a id="타입-악"></a>1.16. 악
교활하고 비열한 싸움 방식을 상징합니다. 2세대부터 에스퍼 타입을 저지하기 위해 추가되었습니다.
### <a id="타입-강철"></a>1.17. 강철
가장 단단한 금속의 타입입니다. 무려 11개 타입을 반감 이하로 받아내는 최고의 방어 타입입니다.

## <a id="타입별-상성"></a>2. 타입별 상성
포켓몬 배틀의 핵심인 상성 관계를 나타냅니다. 4세대 기준 각 타입 간의 시너지와 상호작용을 직관적으로 정리했습니다.

### <a id="상성-방어"></a>2.1. 방어 상성 배율 (특수 무효화 및 주요 피해 관계)
특정 타입 간에는 대미지를 아예 받지 않는 완벽한 무효($0\text{배}$) 상성이 존재합니다. 복합 타입 포켓몬의 경우 두 타입의 상성이 곱해져 최대 $4\text{배}$의 피해를 입거나 $0.25\text{배}$로 대미지가 반감됩니다.

* **완전 무효($0\text{배}$) 공격 관계:**
  * **노말 기술** ➔ 고스트 타입 포켓몬 피격 시 무효
  * **격투 기술** ➔ 고스트 타입 포켓몬 피격 시 무효
  * **전기 기술** ➔ 땅 타입 포켓몬 피격 시 무효
  * **땅 기술** ➔ 비행 타입 포켓몬 피격 시 무효
  * **독 기술** ➔ 강철 타입 포켓몬 피격 시 무효
  * **고스트 기술** ➔ 노말 타입 포켓몬 피격 시 무효

### <a id="상성-공격"></a>2.2. 공격 상성 배율 표

## <a id="스탯"></a>3. 스탯
포켓몬의 육성 수준과 능력치 밸런스를 구성하는 기본 능력값 및 기술 사용 제한 수치를 안내합니다.

### <a id="스탯-hp"></a>3.1. HP (체력)
포켓몬의 생명력 수치입니다. 배틀 중 적의 공격을 받아 체력이 $0$이 되면 기절하며, 기절한 포켓몬은 전투에 참여할 수 없습니다.

### <a id="스탯-공격"></a>3.2. 공격
포켓몬의 물리적인 공격 파워를 의미합니다. '물리 기술'로 분류된 기술을 사용할 때 적에게 가하는 최종 대미지량을 높여줍니다.

### <a id="스탯-방어"></a>3.3. 방어
상대방 포켓몬의 거친 육체적 난타나 타격에 저항하는 능력입니다. 적의 '물리 기술'에 맞았을 때 입는 피해를 경감합니다.

### <a id="스탯-특수공격"></a>3.4. 특수공격
원거리 에너지 방출, 원소 제어, 정신 파동 계열의 능력을 의미합니다. '특수 기술'로 분류된 기술을 다룰 때 위력을 높여줍니다.

### <a id="스탯-특수방어"></a>3.5. 특수방어
상대방이 구사하는 원소 마법이나 에너지 방출 형태의 공격에 저항하는 능력입니다. 적의 '특수 기술'을 허용했을 때 가해지는 최종 대미지를 줄여줍니다.

### <a id="스탯-스피드"></a>3.6. 스피드
배틀 필드에서 행동의 신속성을 결정합니다. 기술 고유의 선제공격 우선순위가 동일하다면, 스피드 수치가 단 1이라도 높은 포켓몬이 무조건 먼저 공격을 실행합니다.

### <a id="스탯-pp"></a>3.7. PP (파워 포인트)
배틀 시 각 개별 기술에 책정된 최대 시전 가능 횟수입니다. 기술을 사용할 때마다 1씩 차감되며, PP가 0이 된 기술은 재충전하기 전까지 발동할 수 없습니다. 모든 기술의 PP가 0이 되면 패널티 기술인 '발버둥'을 사용합니다.

## <a id="특성"></a>4. 특성
포켓몬이 태어날 때부터 가지고 있는 고유의 패시브 능력입니다. 4세대 하트골드/소울실버 시점에서는 한 포켓몬당 최대 2개의 일반 특성 중 하나를 가집니다.

### <a id="특성-일반"></a>4.1. 일반 특성 리스트 (대표 예시)
| 특성명 | 효과 설명 | 대표 포켓몬 |
| :--- | :--- | :--- |
| **맹화** | HP가 전체의 $\frac{1}{3}$ 이하일 때 불꽃 타입 기술의 위력이 1.5배 증가합니다. | 브케인, 파이리 |
| **심록** | HP가 전체의 $\frac{1}{3}$ 이하일 때 풀 타입 기술의 위력이 1.5배 증가합니다. | 치코리타, 이상해씨 |
| **급류** | HP가 전체의 $\frac{1}{3}$ 이하일 때 물 타입 기술의 위력이 1.5배 증가합니다. | 리아코, 꼬부기 |
| **옹골참** | 일격필살 기술(가위자르기, 뿔카노 등)에 당하지 않습니다. (4세대 기준) | 꼬마돌, 롱스톤 |
| **면역** | 독, 맹독 상태이상에 걸리지 않습니다. | 잠만보 |

## <a id="아이템-분류"></a>6. 아이템 분류
하트골드/소울실버의 가방(Bag) 포켓 및 내부 데이터 기준으로 분류된 아이템의 카테고리입니다.

* ### <a id="분류-배틀도구"></a>6.1. 배틀도구 포켓 (기술 | 능력치 | 전용)
  배틀 중 포켓몬에게 지니게 하여 기술 위력을 올리거나 스탯을 보조하는 도구입니다. (예: 생명의구슬, 교정깁스, 전용 장착템)
* ### <a id="분류-육성"></a>6.2. 육성
  포켓몬의 레벨을 올리거나, 기초 포인트(노력치)를 올려주는 도구입니다. (예: 이상한사탕, 타우린, 사키로)
* ### <a id="분류-필드"></a>6.3. 필드
  배틀 밖 필드 이동을 편리하게 돕거나 야생 포켓몬을 퇴치하는 도구입니다. (예: 스프레이, 동굴탈출로프)
* ### <a id="분류-진화"></a>6.4. 진화
  특정 포켓몬을 진화시키는 데 필요한 신비한 돌이나 도구입니다. (예: 번개의 돌, 불꽃의 돌)
* ### <a id="분류-회복"></a>6.5. 회복
  상처 입은 포켓몬의 HP, PP, 상태이상을 치유하는 약품입니다. (예: 풀회복약, 상처약, 해독제)
* ### <a id="분류-몬스터볼"></a>6.6. 몬스터볼
  야생 포켓몬을 포획하는 데 사용되는 볼입니다. (예: 마스터볼, 규토리볼, 하이퍼볼)
* ### <a id="분류-배틀배틀 포켓"></a>6.7. 배틀배틀 포켓
  트레이너 배틀이나 야생 배틀 도중 가방에서 '직접 선택'해 일시적으로 스탯을 높여주는 도구입니다. (예: 이펙트가드, 플러스파워)
* ### <a id="분류-나무열매"></a>6.8. 나무열매
  나무에서 수확하는 열매로, 포켓몬에게 지니게 하면 스스로 회복하거나 배틀 중 가방에서 사용합니다. (예: 자뭉열매, 복분열매)
* ### <a id="분류-머신"></a>6.9. 기술머신(TM) / 비전머신(HM)
  포켓몬에게 새로운 기술을 가르치는 특수 디스크입니다.
* ### <a id="분류-메일"></a>6.10. 메일
  포켓몬에게 지니게 하여 메시지를 적어 다른 트레이너와 교환할 수 있는 편지입니다.
* ### <a id="분류-보물"></a>6.11. 보물
  상점에 높은 가격으로 파는 것 외에는 다른 용도가 없는 귀중품입니다. (예: 금구슬, 별의모래)
* ### <a id="분류-중요물건"></a>6.12. 중요한 물건
  스토리 진행에 필수적이거나 버릴 수 없는 고유의 핵심 도구입니다. (예: 자전거, 실프스코프, GS볼)

## <a id="기술머신-비전머신"></a>7. 기술머신(TM)과 비전머신(HM)
포켓몬에게 인위적으로 기술을 주입하는 장치입니다. 4세대 기준 두 장치의 구동 방식 차이는 다음과 같습니다.

* ### <a id="상세-기술머신"></a>7.1. 기술머신(TM)
  사용하면 사라지는 **일회용** 아이템입니다. 하트골드/소울실버에는 총 92개의 기술머신이 존재합니다.
* ### <a id="상세-비전머신"></a>7.2. 비전머신(HM)
  횟수 제한 없이 **무제한** 사용 가능합니다. 필드에서 장애물을 제거하거나 이동할 때(파도타기, 공중날기 등) 필수적이며, 스토리 진행에 따라 비전신호(배지)를 얻어야 필드 사용이 해금됩니다. 또한 일반적인 방법으로는 기술을 지울 수 없고 검은먹시티의 기술지우기 할아버지를 찾아가야만 지울 수 있습니다.
